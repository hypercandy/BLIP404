# BLIP404
basic project using ```gradle```, ```Spring Boot```, ```Thymeleaf```, and ```GitLab CI```.

## How to run
### 1 - Running the project
From the root directory ...

On Linux/Mac OS:
```shell
./gradlew bootRun
```

On Windows:
```shell
gradlew bootRun
```
