# BLIP404
basic project using ```gradle```, ```Spring Boot```, ```Thymeleaf```, and ```GitLab CI```.

## How to run
From the root directory run `./gradlew bootRun` (or gradlew bootRun on Windows).

The application should run on http://localhost:8080
